Lucene_Net_Demo
==========================================================================================

Lucene.net的示例

### 版本

    - V1.0

### 发起原因

    - 最近项目中有公文管理模块，用到全文检索，利用lucene.net为引擎进行检索。

### 目的

    - 针对中文搜索的示例。

### 语言

    - c#

### 引用框架

    - lucene.net、spring.net

### 入口项目名

    - LuceneNet.Client、LuceneNet.Server

### 测试工程名

    - LuceneNet.ServiceTest

### 主要功能

    - 通过txt文档来产生模拟数据，存入数据库。
    - 自动更新索引。
    - 用户可维护字典。
    - 基于iCatGenerator 生成器来生成数据库的操作代码。
